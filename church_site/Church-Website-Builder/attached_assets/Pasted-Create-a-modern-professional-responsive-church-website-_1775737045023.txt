Create a **modern, professional, responsive church website** for:

Arabic Name (Default): **كنيسة الإيمان**
English Name: **Faith Church**

The website must support **Arabic (RTL) as the default language** and **English as a secondary language**, with a **language switcher in the header**.

The design should be **inspired by the structure and clarity of modern church websites like thesummitchurch.com**, but it must be **original in layout, styling, wording, and branding**.

The goal is to build a **complete digital church platform**, not just a static site.

The site must be **responsive, interactive, organized, and capable of automatically importing content from Word and Excel documents**.

---

# BRANDING

Use the uploaded **Faith Church logo** as the core identity.

Logo elements represent:

Blue → faith, stability, trust
Orange → hope, welcome, energy
Green → growth, life, community

Design guidelines:

Primary color: Blue
Accent color: Orange
Secondary accent: Green
Background: White or very light grey
Text: Dark gray / navy

Design style:

Modern
Minimal
Warm
Clean
Spiritual but not overly decorative

Use:

• large spacing
• rounded cards
• subtle shadows
• modern typography
• soft section transitions

The cross from the logo represents the spiritual center of the church.

# DOCUMENT IMPORT SYSTEM

The website must include a **smart document reader**.

Church staff will upload:

Word files (.docx)
Excel files (.xlsx)

The system should:

Read files
Extract text and tables
Suggest page placement

Examples:

Word files → announcements, ministry descriptions, sermons, policies

Excel files → events lists, schedules, directories

Workflow:

Upload document
AI reads structure
Preview content
Select page placement
Publish
